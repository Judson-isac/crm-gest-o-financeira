# **App Name**: Request Reader

## Core Features:

- HTTP Request: Make HTTP requests to a specified URL.
- HTML Display: Render and display the HTML content received from the HTTP request.
- URL Input: Allow users to input the URL for the HTTP request.
- Error Handling: Implement error handling to gracefully manage issues such as invalid URLs or network errors.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) to evoke a sense of trust and reliability.
- Background color: Light gray (#F0F4F8) to ensure readability and focus on content.
- Accent color: Soft purple (#A259FF) for interactive elements and visual cues.
- Body and headline font: 'Inter' for a clean, modern and readable interface.
- Use simple, clear icons to represent actions such as 'Send Request' or 'View Source'.
- Implement a clean, straightforward layout with URL input at the top, followed by the rendered HTML content.
- Subtle loading animations while fetching the HTTP content.
